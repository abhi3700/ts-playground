# Typescript Playground

## Description

This is for typescript code.

## Install

```sh
yarn
```

## Build

```sh
yarn build
```

## Test

```sh
yarn test
```

## Coverage

```sh
yarn coverage
```

## Lint

```sh
yarn lint
```

## Format

```sh
yarn format
```

## Clean

```sh
yarn clean
```

## Clean All

```sh
yarn clean:all
```
